The primary purpose a reference manager is to store metadata for later use within a document. These reference managers try to integrate with document preparation tools such as LaTeX so that citations are easy to format without having to manually enter in every piece of information su.

List of reference managers

  • docear
  • endnote
  • jabref
  • mendeley
  • paperpile
  • paperpile-older
  • papers
  • publish-or-perish
  • qiqqa
  • readcube
  • zotero
