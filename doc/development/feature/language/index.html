SVG

You can embed text strings in multiple languages inside of SVG. It can be made to display the one used by the system.

Example: https://commons.wikimedia.org/wiki/File:SystemLanguage.svg.

I was looking into this because this diagram on Wikipedia was using the German "pyruvat" instead of the English "pyruvate" https://en.wikipedia.org/wiki/File:Phosphotransferase_system.svg via https://en.wikipedia.org/wiki/PEP_group_translocation.
